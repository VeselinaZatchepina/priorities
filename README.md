# Priorities

<p>Приложение даёт возможность узнать Ваши жизненные приоритеты. Вы просто вводите все важные (именно для Вас) категории, а приложение помогает вычислить какие ТРИ из всех категорий имеют наибольшую ценность для Вас.</p>

***Приложение написано на Flutter***
